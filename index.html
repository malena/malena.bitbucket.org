I also understand and have set up projects using task managers to compile and minify HTML, CSS and JS. Lately I've been looking more into webpack and npm scripts.

I am quick at prototyping, using bootstrap or skeleton, and follow the SMACSS methodologies to organize CSS.

I am a master of CSS and currently mastering JavaScript. I have used a vast number of libraries like jQuery, or frameworks like Backbone and Angular, but I would like to strengthen writing pure JavaScript.

These days I’ve been diving into React.js and would love to find an opportunity to work in an environment where I can master building UI components.

My roots as a web developer

My first web job I was simply styling CSS components within Joomla and populating content.

My second job was part of a software team where I was known as the CSS cross browser magician. Back then, ie6 was the most used browser. I moved up to create various layouts as per design specifications and manage the front end entirely, including doing research on user experience and monitoring video recorded usability tests.

Once I learned about the power of JavaScript, I slowly started writting my own scripts. I learned to use jQuery but developed a better programming foundation once I went back to school for a JavaScript course.

Eventually I worked on dev team that introduced me to “best practices”, and with some mentorship and encouragement I became a front end lead of several projects.
